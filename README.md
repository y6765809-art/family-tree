## 🌟 תכונות

### ✅ בניית עץ משפחה מלא
- **הוספת הורים** - בנה את העץ למעלה
- **הוספת ילדים** - בנה את העץ למטה  
- **בני/בנות זוג** - הוסף בן/בת זוג לכל אדם
- **מידע מפורט** - שם, שנים, תיאור, הערות
- **סימון צדיקים** - סימון מיוחד לצדיקים ואישים חשובים

### 💾 שמירה וגיבוי
- **שמירה מקומית** - localStorage אוטומטי
- **ייצוא JSON** - שמור לקובץ JSON
- **טעינה מקובץ** - טען עץ ישן
- **גיבוי לגוגל דרייב** - העלה את העץ לדרייב ☁️
- **ניהול עצים מרובים** - נהל כמה עצים בו זמנית

### 📄 ייצוא והדפסה
- **ייצוא PDF** - שמור את העץ כ-PDF
- **הדפסה** - הדפס ישירות
- **רשימת דורות** - רשימה מסודרת לפי דורות

### 🎨 ממשק משתמש
- **עברית מלאה** - כל הממשק בעברית
- **זום** - הגדל/הקטן את העץ
- **עריכה קלה** - ערוך כל פרט בקלות
- **מחיקה** - מחק פריטים בקלות
- **גרירה** - גרור את העץ למקום נוח

## 📖 מדריך שימוש

### 🆕 יצירת עץ חדש
1. לחץ על **"➕ התחל עץ יוחסין"**
2. הזן שם, שנים (אופציונלי), תיאור
3. לחץ **"צור אדם ראשון"**

### 👥 הוספת הורים
1. לחץ על **אדם בעץ**
2. הפאנל הימני יפתח
3. לחץ **"➕ הוסף הורה"**
4. מלא פרטים → **"הוסף הורה"**

### 👶 הוספת ילדים
1. לחץ על **אדם בעץ**
2. בפאנל הימני לחץ **"הוסף ילד"**
3. מלא פרטים → **"הוסף ילד"**

### 💑 הוספת בן/בת זוג
1. לחץ על **אדם בעץ**
2. בפאנל הימני לחץ **"הוסף בן/בת זוג"**
3. מלא פרטים → **"הוסף"**

### 💾 שמירה
#### שמירה מקומית (זמני)
- הכל נשמר אוטומטית ב-localStorage

#### שמירה קבועה (מומלץ!)
1. לחץ **"💾 שמור"**
2. הקובץ `yichus_tree.json` יורד
3. **שמור במקום בטוח!**

#### שמירה לגוגל דרייב
1. לחץ **"☁️ שמור לדרייב"**
2. הקובץ יורד
3. העלה ל-[Google Drive](https://drive.google.com)

### 📂 טעינת עץ ישן
1. לחץ **"📂 טען"**
2. בחר את קובץ ה-JSON
3. העץ יטען!

### 📄 ייצוא PDF
1. לחץ **"📄 ייצא PDF"**
2. בחר הגדרות (גודל, כיוון)
3. לחץ **"💾 שמור PDF"**
